# WeatherApp


Weather App ☀️☁️️
This repository contains the code for a weather app built as part of Task 1 during my internship at Bharat Intern. The app demonstrates proficiency in using HTML, CSS, and JavaScript for front-end development.

Key Features:

Dynamic weather information: Retrieves and displays real-time weather data for a user-specified location using a weather API.
Responsive design: Adapts seamlessly to different screen sizes and devices.
Intuitive user interface: Provides a clear and user-friendly way to interact with the app's features.
Clean and maintainable code: Follows best practices for code organization and documentation.

Technologies Used:
Front-End: HTML, CSS, JavaScript
Weather API: Open weather API
